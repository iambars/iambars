## 🚀 Projects

### **WorkPulse**
https://workpulse-iambars.vercel.app

A personal project that aims to simplify attendance tracking and shift management for individuals and teams, making work hours more structured and human-friendly.

- User authentication with OAuth (Google) and credential-based sign-in  
- Set and manage custom shift schedules  
- Automatically generate daily attendance based on selected shifts  
- Compute total work hours and overtime with an automatic 1-hour lunch break deduction  

**Future enhancements:**
- Organization structure (teams, roles, and permissions)  
- Admin dashboard for reporting and analytics  
- Advanced attendance reports and summaries  
- Employee performance and productivity insights  
- Exportable timesheets and reports (PDF/CSV)
